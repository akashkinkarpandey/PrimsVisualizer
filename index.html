This tool will be helpful to visualize the Prims MST Algorithm. While a graph has been provided you can generate a new graph with the number of nodes desired, by writting an input and clicking generate graph.

PRIMS ALGORITHM

  1. Determine an arbitrary vertex as the starting vertex of the MST.
  2. Follow steps 3 to 5 till there are vertices that are not included in the MST (fringe vertex).
  3. Find edges connecting any tree vertex with the fringe vertices.
  4. Find the minimum among these edges.
  5. Add the chosen edge to the MST if it does not form any cycle. Return the MST and exit.
